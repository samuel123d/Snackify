# Snackify | Loja Virtual

O Snackify é uma plataforma de e-commerce simplificada, focada na venda de salgadinhos e bebidas. O projeto foi desenvolvido para oferecer uma experiência de compra ágil, integrando um catálogo de produtos a um sistema de gestão de dados em tempo real.

**Link de Acesso:** [Visualizar Projeto](https://samuel123d.github.io/Snackify)

---

## Resumo do Projeto

A aplicação permite a navegação por categorias de snacks e bebidas, adição de itens ao carrinho e finalização de pedidos. O grande diferencial técnico é a persistência e gestão de dados dinâmica, garantindo que o inventário e as informações do sistema estejam sempre atualizados.

### Tecnologias e Ferramentas

| Categoria | Tecnologias Utilizadas |
| :--- | :--- |
| **Frontend** | HTML5, CSS3, JavaScript (ES6+) |
| **Frameworks** | Tailwind CSS / Bootstrap (Interface) |
| **Banco de Dados** | Firebase Realtime Database / Firestore |
| **Autenticação** | Firebase Auth |

---

   Estrutura de Diretórios

A organização do repositório reflete uma arquitetura voltada para a integração com serviços Cloud:

assets/: Identidade visual, ícones e estilização global.
src/: Scripts de lógica de negócio e manipulação do DOM.
js/firebase-config.js: Configuração e inicialização dos serviços Firebase.
index.html: Interface principal da loja e vitrine de produtos.

---

   Funcionalidades Principais

O projeto destaca competências em integração de serviços BaaS (Backend as a Service):

>   Gestão via Firebase
> Sincronização em tempo real de produtos e pedidos, utilizando o ecossistema Google Cloud para garantir escalabilidade e segurança mínima.
